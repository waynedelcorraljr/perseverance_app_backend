# README

### Ruby version
- 2.6.6

### Rails version
- 6.0.3

## Setup instructions

### Local Install
- clone repo
- cd root directory

### Database creation
- run `rails db:create`

### Database initialization
- run `rails db:migrate`

### Deployment instructions
- run `rails server`
- visit http://localhost:3000/api/v1/earthdates and http://localhost:3000/api/v1/photos
